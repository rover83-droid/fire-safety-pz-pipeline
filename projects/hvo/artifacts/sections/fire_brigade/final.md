# ПРОЕКТНЫЕ РЕШЕНИЯ ПО ОБЕСПЕЧЕНИЮ БЕЗОПАСНОСТИ ПОДРАЗДЕЛЕНИЙ ПОЖАРНОЙ ОХРАНЫ ПРИ ЛИКВИДАЦИИ ПОЖАРА

## 8.1 Общие проектные решения

| Показатель | Значение |
|---|---|
| Объект | здание химводоочистки на территории производственной площадки «Северный луч» |
| Класс функциональной пожарной опасности | Ф5.1 |
| Назначение | производственное здание химводоочистки |
| Габариты в осях | 78 x 36 м |
| Ширина здания | 36 м |
| Высота до верха парапета | 12,18 м |
| Площадь здания | 3042 м2 |
| Строительный объем | 32678,8 м3 |
| Этажность | 1 этаж + антресоль, без подвала |
| Встройка | 2 этажа |
| Степень огнестойкости | III |
| Класс конструктивной пожарной опасности | С0 |
| Предварительная категория здания | В |
| Противопожарные системы | СПС, СОУЭ, ВПВ, НПВ |

Для здания химводоочистки класса функциональной пожарной опасности Ф5.1 предусматриваются решения, обеспечивающие деятельность подразделений пожарной охраны при ликвидации пожара: пожарные проезды и подъездные пути к зданию для мобильных средств пожаротушения, средства подъема личного состава на кровлю, а также противопожарный водопровод, включая предусмотренные исходными данными системы ВПВ и НПВ.

Здание принято производственным, одноэтажным с антресолью, без подвала, с двухэтажной встройкой; высота до верха парапета составляет 12,18 м, площадь здания 3042 м2, строительный объем 32678,8 м3, степень огнестойкости III, класс конструктивной пожарной опасности С0, предварительная категория по пожарной опасности В. Расчетный расход воды на наружное пожаротушение производственного объекта определяется по зданию или сооружению, требующему наибольшего расхода воды, по таблицам СП 8.13130.

## 8.2 Подъезды, проезды и площадки для пожарной техники

С учетом ширины здания 36 м подъезд мобильных средств пожаротушения к зданию предусматривается с двух сторон, поскольку для зданий класса Ф5 при ширине более 18 м требуется подъезд с двух сторон.

Минимальная ширина пожарных проездов принимается не менее 3,5 м, так как высота здания до верха парапета составляет 12,18 м и относится к диапазону до 13 м включительно.

Расстояние от края проезжей части или укрепленной спланированной поверхности, обеспечивающей проезд пожарных автомобилей, до стен здания принимается в пределах 5-8 м, поскольку высота здания составляет более 12 м, но не более 28 м.

## 8.3 Доступ к наружному противопожарному водоснабжению

На территории производственного объекта предусматриваются источники наружного противопожарного водоснабжения для тушения пожара; исходными данными для здания химводоочистки наружное противопожарное водоснабжение принято предусмотренным.

При разработке схемы НПВ расчетный расход воды на наружное пожаротушение здания химводоочистки определяется по требованиям СП 8.13130 для производственных объектов с учетом строительного объема 32678,8 м3, степени огнестойкости III, класса конструктивной пожарной опасности С0 и окончательной категории здания по пожарной опасности.

## 8.4 Выходы на кровлю, пожарные лестницы и ограждения

Так как высота здания до верха парапета составляет 12,18 м, для здания предусматриваются выходы на кровлю с лестничных клеток непосредственно или через чердак, либо по лестницам 3-го типа, либо по наружным пожарным лестницам.

Для здания класса Ф5 количество выходов на кровлю принимается не менее одного; расположение выходов на кровлю назначается с учетом требования устройства пожарных лестниц через каждые 200 м по периметру зданий класса Ф5.

При устройстве наружной пожарной лестницы для подъема на кровлю здания высотой 12,18 м принимается лестница типа П1; лестницы типа П2 требуются для подъема на высоту более 20 м либо при перепадах высоты кровли более 20 м.

Пожарные лестницы выполняются из негорючих материалов, размещаются не ближе 1 м от окон и принимаются в конструктивном исполнении, обеспечивающем возможность передвижения личного состава подразделений пожарной охраны в боевой одежде и с дополнительным снаряжением.

Для производственного здания выходы на кровлю принимаются по требованиям СП 4.13130; данное требование учитывается при разработке архитектурно-строительных решений кровли здания химводоочистки.

Для обслуживания покрытия производственного здания и обеспечения пожарного доступа предусматривается выход на кровлю; конкретный тип выхода и его привязка уточняются архитектурно-строительными чертежами кровли.

## 8.5 Средства подключения мобильной пожарной техники к системам ВПВ/сухотрубам

Внутренний противопожарный водопровод исходными данными предусмотрен; при применении сухотрубов для подключения мобильной пожарной техники сухотруб выполняется как не заполненный в дежурном режиме водой вертикальный трубопровод ВПВ с пожарными запорными клапанами на этажах или полуэтажах и фасадной соединительной головкой для внешнего источника водоснабжения.
